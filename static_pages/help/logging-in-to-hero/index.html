Help/Logging In to Hero
SummaryInstructions on how to log in to the Hero platform. The Hero platform is designed to make creating and managing lists of links more efficient than ever. To use the platform, users must first log in to their accounts. Logging in to Hero is a straightforward process that requires only a few steps.
The Hero platform is a modern website designed to make creating and managing lists of links more efficient than ever. To use the platform, users must first log in to their accounts. Logging in to Hero is a straightforward process that requires only a few steps.

The first step is to go to the website’s main page. On the top right-hand corner of the page, there will be a button with the icon of a person. Clicking this will take the user to a page where they can enter their username and password. Once the user has entered their credentials, they can log in to their account.

Once the user has logged in, they will be directed to their profile page. This page will display any lists the user has created and any Hero Space, which is a collection of lists of links. From here, the user can begin creating and managing their lists.

It is also important to note that users can log in to their accounts from any device, including mobile phones and tablets. To do this, the user must first download the Hero app from the iTunes Store or Google Play. Once the app is installed, the user can open it and log in using the same credentials used for the website.

Logging in to the Hero platform is an easy process. Following the steps above, users can quickly and easily access their accounts to create and manage their lists.

Can users log in to their accounts on mobile devices?

Yes, users can log in to their accounts on mobile devices by downloading the Hero app from the iTunes Store or Google Play.

What are Hero Spaces?

Hero Spaces are collections of lists of links.

What can users do once they have logged in to their accounts?

Once they have logged in, users can begin creating and managing their lists.

What credentials are required to log in to the app?

The same credentials used for the website must be used to log in to the app.

What information is required to log in to an account?

The user must enter their username and password to log in to their account.

What is displayed on the profile page?

The profile page displays any lists the user has created and any contributions the user has made toward other users' Hero Spaces.

Where can users download the Hero app?

The Hero app can be downloaded from the Apple App Store or Google Play.

Where can users find the login button on the website?

The login button can be found in the top right-hand corner of the website's main page.

Where will the user be directed after logging in?

The user will be directed to their profile page, which will display any lists the user has created and any Hero Space.
